OPINION 2019 is a project in which we are able to analyze the opinions of public on about the Lok Sabha elections of 2019 and group them and Show what public thinks about the two main parties participating in the Elections.


In OPINION 2019 we will analyze the views of public written on a website and on Android Application made by us, with help of Sentimental Analysis which will be Written by us using Machine Learning.Then we will count the percentage of negative views and positive views about a particular party which will be contesting elections
